# G5.2 + 斜杠空格 — Skills 深度对齐 Grok

**日期**: 2026-07-16  
**状态**: 已完成（2026-07-16）  
**Grok 源码/文档**: `08-skills.md` · `prompt/skills.rs` · `04-slash-commands.md`

## 问题

1. **空格被吞**：`filterable` 选项浮层（斜杠命令 / 技能切换器）把 Space 当成 sessions 一样丢弃，无法键入多词过滤。
2. **Grok 能力未贯彻**：skill 路径仅 `.qling`/`skills`；未默认扫 `.claude`/`.cursor`/`.agents`/`.grok`；`/name args` 丢弃参数。

## 修复与对齐

| 项 | 行为 |
|----|------|
| Space | filterable options → `handleChar(" ")` + refilter |
| `/skill` 切换器 | `filterable: true`，键入（含空格）检索 |
| 多词过滤 | token 全匹配 id/label/desc |
| 扫描路径 | cwd 优先 → user → bundled；含 `.grok/.agents/.claude/.cursor` skills(+commands) |
| 开关 | `QLING_*_SKILLS_ENABLED=0` 关闭对应 vendor |
| `/skill-name args` | 加载 skill 并前缀展示参数 |

## 非目标

- Grok marketplace / plugin install
- 自动 model 调用 skill（disable-model-invocation 全解析）
- 递归多级嵌套 skills 目录（保持一层 SKILL.md）
