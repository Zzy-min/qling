# `qingling` 阶段 B：Slash 中文别名补齐规格（2026-05-17）

## 背景

- 重做计划要求英文主命令优先，同时保留中文别名以降低迁移成本。
- 当前 `/goal`、`/loop`、`/tasks` 尚未提供中文别名入口。

## 目标

为以下命令补齐等价中文别名：

- `/goal` -> `/目标`
- `/loop` -> `/循环`
- `/tasks` -> `/任务`

## 范围

- 仅补别名映射，不改变原命令语义。
- 保持英文命令为主，中文别名作为兼容入口。

## 验收

- 单测覆盖中文别名调用路径并通过。
- `npm run build`、`npm run ci:check` 通过。
