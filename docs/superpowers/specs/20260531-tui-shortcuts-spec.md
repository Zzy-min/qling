# `qingling` TUI 快捷键帮助规格（2026-05-31）

## 背景

轻灵 TUI 已支持多行输入、历史检索、状态线和长任务反馈，但快捷键目前只散落在 header 和代码中。为了接近 Claude Code 的可发现交互体验，用户需要在会话内随时查询快捷键，而不必重启或翻文档。

## 目标

- 新增 `/shortcuts` slash command，中文别名 `/快捷键`。
- 展示当前 TUI 支持的快捷键与行为。
- 明确哪些能力是本地输入编辑，不上传、不持久化。
- `/help` 中列出 `/shortcuts`。
- 只新增帮助展示，不改变键盘处理逻辑。

## 非目标

- 不新增新的键盘快捷键。
- 不实现可配置快捷键。
- 不持久化快捷键偏好。
- 不读取消息正文或外部文档。

## 行为

- `/shortcuts` 输出中文表格式帮助。
- 包含至少以下条目：
  - `Enter` 发送当前输入。
  - `Ctrl+N` 插入换行。
  - `Ctrl+R` 搜索当前会话内历史。
  - `Ctrl+C` 清空当前输入。
  - `↑/↓` 切换历史。
  - `←/→` 移动光标。
  - `Ctrl+O` 预留用于长输出展开。
- 报告末尾说明：快捷键只作用于本地 TUI 输入缓冲，不上传输入历史。

## 验收

- slash 单测覆盖 `/shortcuts` 与 `/快捷键`。
- `/help` 包含 `/shortcuts`。
- `npm run build`、相关单测和 `npm run ci:check` 通过。
