# Spec: Phase 1.3 — TUI 产品化

**日期**: 2026-07-09  
**状态**: Accepted  
**父路线**: 竞品路线图 Phase 1.3（对标 OpenCode / Crush / Pi）

---

## 1. 目标

| ID | 目标 | 验收 |
|----|------|------|
| T1 | Top bar 显示 sessionMode + permissionMode | formatTopBar 单测 |
| T2 | Statusline 显示 模式=plan\|agent 且真正打印到 TUI | showPrompt 输出含 statusline |
| T3 | 工具输出卡片折叠可复用、可测 | formatToolOutputCard 单测 |
| T4 | 输入区/工具输出批量 write 减闪烁 | 逻辑合并为单次 write |
| T5 | `/expand` 切换长输出展开（与 Ctrl+O 同源） | slash + 状态反馈 |

### 非目标

- 完整差分渲染引擎（Pi 级）  
- 可滚动 viewport 虚拟列表  
- 主题系统  

---

## 2. 设计摘要

1. **chrome**：`formatTopBar` 增加 `Mode` / `Perm` 字段。  
2. **statusline**：`sessionMode` 字段；`StreamUI.showPrompt` 在启用时打印 statusline。  
3. **tool card**：`formatToolOutputCard(output, { expand })` 统一折叠语义。  
4. **expand**：`StreamUI.toggleExpandLongToolOutput()` + slash `/expand`。  
5. **batch write**：`writeInputValue` / `printToolOutput` 拼串后一次 `stdout.write`。
