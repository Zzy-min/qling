## 2. 设计

### 2.1 package.json

- `engines.node`: `>=18`  
- `license`, `author`, `homepage`, `repository`, `bugs`  
- `keywords`: agent, cli, tui, local-first, chinese, …  
- `files`: 发布所需最小集合（dist / skills / README* / LICENSE / CHANGELOG）  
- 保留 `bin.qling` → `./dist/index.js`  

### 2.2 文档

- `README.en.md`：定位、安装、最小配置、4 命令、隐私边界  
- `docs/install.md`：npm / git / bootstrap / Windows PowerShell / scoop 草案 / winget 草案  
- `packaging/scoop/qling.json`、`packaging/winget/*.yaml` 为**草案**，标注需 release 资产  

### 2.3 Onboarding

- 导出 `buildOnboardingSteps()` / `formatOnboardingCard()`  
- 3 步：发任务、`/` 命令、`/doctor` + `/privacy`  
- 仍写入 `~/.qling/.onboarded`；非 TTY 跳过
