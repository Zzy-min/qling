## 2. 设计

### 2.1 Eval smoke

- `src/eval/`：任务定义 + runner  
- 任务类型：纯函数/工具边界（sandbox、sensitive path、network mode、presets、onboarding）  
- 输出：pass/fail 计数 + 每任务状态  
- 脚本：`scripts/eval-smoke.mjs` → `npm run eval:smoke`  
- 不依赖真实 API key  

### 2.2 Skills

- 约定：目录下 `SKILL.md` 或 `index.md` + YAML frontmatter（已有）  
- 模板：`skills/templates/SKILL.md`  
- 示例：`skills/examples/repo-triage/SKILL.md`  
- 文档：`docs/skills.md`  

### 2.3 MCP presets + store

- `src/mcp/presets.ts`：内置 preset（memory / fetch / filesystem / git 等常见 npx 命令草案）  
- `src/mcp/store.ts`：`~/.qling/mcp-servers.json` 读写  
- CLI：`qling mcp` | `list` | `presets` | `add <id>` | `remove <name>`  
- 加载：config 应用时合并 store（env `QLING_MCP_SERVERS` 仍优先）  
- slash `/mcp` 增加 list 提示  

### 2.4 SDK

- `src/sdk.ts` 导出 AgentLoop、config helpers、presets 解析、write sandbox helpers  
- package.json `exports` 字段  
- `docs/sdk.md` 最小示例
