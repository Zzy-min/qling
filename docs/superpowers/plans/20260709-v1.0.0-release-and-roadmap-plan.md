## 1. 版本语义说明

| 字段 | 值 |
|------|-----|
| 旧版本 | `0.5.0` |
| 新版本 | `1.0.0` |
| 含义 | 核心工作台契约进入稳定；本地优先中文 Agent CLI 可日常使用 |
| 不意味着 | 已追平 OpenCode 生态或功能完备 |
